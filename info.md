# Marstek Cloud Battery

Monitor your Marstek battery system in Home Assistant.

## Features

- **Real-time battery monitoring** – SOC, charge/discharge power, load
- **Automatic token management** – No manual API handling needed
- **Multiple batteries** – Support for multiple devices with aggregated sensors
- **Configurable polling** – 10-3600 seconds update interval

## Quick Start

After installation via HACS:
1. Go to **Settings → Devices & Services → Add Integration**
2. Search for **Marstek Cloud Battery**
3. Enter your Marstek cloud credentials
4. Configure scan interval (default: 60s)

## Requirements

- Home Assistant 2024.1.0+
- Marstek cloud account
- EU-based Marstek battery system
